# Steam Games Success Analysis

## Project Overview
#### This project analyzes the Steam Games dataset to identify the factors that influence game success. The analysis was conducted using Python for data preprocessing and machine learning, and Power BI for interactive dashboards.

## Dataset
#### The dataset used in this project was obtained from Kaggle and contains comprehensive information about games available on the Steam platform. The dataset contains:
#### - Game Name
#### - Price
#### - Genres
#### - Developers
#### - Publishers
#### - Positive Reviews
#### - Negative Reviews
#### - Peak Concurrent Users (CCU)
#### - Estimated Owners
#### - Playtime
#### - Metacritic Score
#### - Release Date

### Kaggle: https://www.kaggle.com/datasets/artermiloff/steam-games-dataset
